# Citi Private Bank landing page

A bespoke single-page brief prepared for Citi Private Bank, for use at eMerge Americas Miami, Booth 836. The page carries the Eduba fit analysis written in the bank's own institutional register, anchored to the Client 360 / CitiScribe / AskWealth CIO rollout timing. Brand tokens come from the Citi Private Bank brand pack: Citi navy (#003B70), mid-blue link (#056DAE), champagne-gold accent (#B38E5D), warm stone (#F5F2EC), Interstate typography with Helvetica Neue and Arial fallbacks. No JavaScript. Mobile responsive to 375px. Logo sits top-left, Eduba attribution appears only in the footer.

Intended URL path: /for/citi-private-bank

Files:
- `index.html` - page structure and copy
- `styles.css` - token-only CSS
- `logo.svg` - Citi Private Bank mark (from Brand Pack/assets)

CTA resolves to `https://calendly.com/thecro-eduba/30min` (Matt Creamer, Eduba's CRO).
